# Privacy Policy for JingZen Translate

**Last updated: 2026-07-08**

## Data Collection

JingZen Translate does **not** collect, store, or transmit any personal information to the extension developer or any third-party servers controlled by the developer.

## What Data Is Processed

### API Key
Your API key (for services like DeepSeek, OpenAI, etc.) is stored **locally** in your browser using `chrome.storage.local`. It is never sent anywhere except directly to the API endpoint you configure.

### Web Page Text
When you use the translation feature, text from the web page you are viewing is sent to the third-party API service you have configured (e.g., DeepSeek, OpenAI). The extension developer does **not** have access to, control over, or responsibility for how these third-party services handle your data.

Please refer to the privacy policy of your chosen API provider:
- DeepSeek: https://platform.deepseek.com/privacy
- OpenAI: https://openai.com/policies/privacy-policy
- Other providers: consult their respective privacy policies

## Data Storage

All configuration data (API key, language preferences, provider settings, translation cache) is stored exclusively in your browser's local storage (`chrome.storage.local`). This data can be cleared at any time by:
- Using the "Reset All Settings" button in the extension options page
- Uninstalling the extension

## Third-Party Services

This extension connects to API endpoints that **you** configure. No data is sent to any server operated by the extension developer.

## Tracking & Analytics

This extension does **not** include any analytics, tracking, or advertising code.

## Contact

For questions about this privacy policy, please open an issue on the extension's GitHub repository.

## Changes

This privacy policy may be updated from time to time. Any changes will be reflected in the updated extension listing.
